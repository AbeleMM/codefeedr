The GHTorrent plugin is mainly focused on mirroring the [original GHTorrent project](http://www.ghtorrent.org/) in CodeFeedr.

## Installation

```scala
dependencies += "org.codefeedr" %% "codefeedr-ghtorrent" % "LATEST_VERSION"
```
